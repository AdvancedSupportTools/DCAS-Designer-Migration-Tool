# DCAS-Designer-Migration-Tool
Designer ECY1_2 Migration Tool 1.0.0 is a community-developed PowerShell/WPF utility that helps analyze Designer package ZIP files, inspect DG5 content, extract data point paths, discover BACnet objects from ECLYPSE controllers, automatically map replacement paths, and generate updated DG5 files to streamline Gen 1 to ECLYPSE migration workflows.
